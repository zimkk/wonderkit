# Skill: Summarize

**Agent:** Any (Researcher, Writer, Orchestrator)
**Trigger:** User asks for a summary, TL;DR, or condensed version of content.

## Instructions

1. Read the full input content.
2. Identify the 3-5 most important points.
3. Write a summary in the following format:

```
## TL;DR
[1-2 sentences capturing the core message]

## Key points
- [Point 1]
- [Point 2]
- [Point 3]

## Details (optional)
[Any nuance worth preserving for the requester's context]
```

4. Match the length of the summary to the length of the input (1-page input → 1 paragraph summary; book → multi-section summary).
5. Do not add opinions or interpretations — summarise what was actually said.
